# Flask 个人博客系统

一个基于 Python Flask 构建的完整个人博客系统，支持 Markdown 写作、代码高亮、文章搜索等功能。

## 功能特性

- 📝 文章列表页（首页），按发布时间倒序展示
- 📖 文章详情页，支持 Markdown 渲染和代码高亮
- 🔐 管理员登录（单用户，配置文件配置）
- ✏️ 后台管理：发布文章、编辑文章、删除文章（需登录）
- 🔍 文章标题搜索功能
- 📄 分页功能
- 👁️ Markdown 实时预览
- 📱 响应式设计，支持移动端

## 技术栈

- **后端**: Python 3.8+, Flask 3.0
- **数据库**: SQLite + SQLAlchemy ORM
- **前端**: Bootstrap 5.3 + Bootstrap Icons
- **Markdown 渲染**: Python-Markdown + highlight.js
- **表单验证**: Flask-WTF + WTForms
- **用户认证**: Flask-Login

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行项目

```bash
python app.py
```

首次运行时，系统会自动：
- 创建 SQLite 数据库文件 `blog.db`
- 创建文章表
- 插入 3 篇示例文章

### 3. 访问应用

打开浏览器访问：http://localhost:5000

## 功能验证步骤

### 1. 浏览文章
1. 访问 http://localhost:5000
2. 查看首页文章列表，应显示 3 篇示例文章
3. 点击任意文章标题，进入文章详情页
4. 验证 Markdown 格式和代码高亮是否正常显示

### 2. 搜索文章
1. 在顶部搜索框输入关键词（如 "Flask"）
2. 点击搜索按钮或按回车
3. 验证搜索结果是否正确

### 3. 管理员登录
1. 点击右上角 "登录" 按钮
2. 输入用户名 `admin`，密码 `admin123`
3. 点击登录，验证是否成功跳转到后台管理页面

### 4. 发布新文章
1. 登录后，点击导航栏 "写文章" 或后台的 "发布新文章" 按钮
2. 输入文章标题和内容（支持 Markdown）
3. 点击 "预览" 标签页查看实时预览
4. 点击 "发布" 按钮
5. 验证新文章是否出现在首页列表

### 5. 编辑文章
1. 在后台管理页面，点击任意文章的编辑按钮（铅笔图标）
2. 修改文章内容
3. 点击 "发布" 保存修改
4. 验证文章内容是否已更新

### 6. 删除文章
1. 在后台管理页面，点击任意文章的删除按钮（垃圾桶图标）
2. 在确认弹窗中点击 "确认删除"
3. 验证文章是否已从列表中移除

### 7. 退出登录
1. 点击右上角 "退出" 按钮
2. 验证是否成功退出并返回首页

## 配置说明

可以通过环境变量或修改 `config.py` 来配置：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `SECRET_KEY` | Flask 密钥 | `dev-secret-key-change-in-production` |
| `DATABASE_URL` | 数据库连接地址 | `sqlite:///blog.db` |
| `ADMIN_USERNAME` | 管理员用户名 | `admin` |
| `ADMIN_PASSWORD` | 管理员密码 | `admin123` |

## 生产环境部署

> **警告**: 生产环境请务必修改默认的 `SECRET_KEY` 和管理员密码！

推荐使用 Gunicorn + Nginx 部署：

```bash
# 安装 Gunicorn
pip install gunicorn

# 启动应用
gunicorn -w 4 -b 0.0.0.0:5000 app:app
```
